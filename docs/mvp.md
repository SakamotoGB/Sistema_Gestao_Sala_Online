## MVP
# Funcionalidades Essenciais:
- Feature de adicionar nota ;
- Feature criação de avaliação e atividades;
- Feature para anexo de material de apoio;
- Feature criação das turmas e a opção de se juntar a elas;
- Feature opção de login e cadastro;
- Feature prazo das atividades;
- Feature de entrega de atividades

# Funcionalidades Futuras:
- Feature de notificações
- Feature supervisão dos estudantes para os responsáveis
- Feature de feedback e comentários
- Feature pendência das atividades
- Feature de fórum de Dúvidas

# Justificativa da priorização:
As features selecionadas para o MVP representam a infraestrutura mínima do sistema. Elas garantem o fluxo educacional básico: O cadastro dos usuários é realizado, o professor consegue agrupar alunos, propor materiais de apoio, atividades e avaliações, o aluno consegue responder e o professor consegue quantificar o resultado. Enquanto o resto das features descritas como futuras, dizem mais a respeito da qualidade da experiencia do usuario
