# USER STORIES

## US01- Criação da turma virtual
Como professor, quero criar uma turma virtual para agrupar meus alunos e os materiais de uma disciplina.
 - Apenas usuários com o perfil “professor” podem criar uma turma.
 - Ao gerar a turma, o sistema deve gerar um código de convite.
 - O professor pode convidar os alunos para a turma, no processo de criação.

## US02- Prazo da Atividade
Como professor, quero adicionar um prazo às atividades, para ter controle sobre a data de entrega das atividades.
 - Durante a criação de uma atividade, o professor pode selecionar uma data e horário específicos em um calendário de horas.
 - O prazo definido deve ficar visível tanto para o professor quanto para os alunos vinculados àquela turma.
 - O sistema deve permitir que o aluno envie a atividade após o prazo, mas deve marcá-la com um rótulo visual.

## US03- Avaliação e Feedback Privado 
Como professor, quero adicionar um comentário particular a uma atividade entregue.
 - O professor deve ter um campo de texto disponível na tela de visualização da entrega individual do aluno.
 - O comentário deve ser privado, visível apenas para o professor que o criou e para o aluno que realizou a entrega.
 - O aluno deve receber uma notificação de que um novo comentário particular foi adicionado.

## US04- Pendências e Prazos
Como estudante, desejo que visualizar uma lista de todas as atividades pendentes e seus prazos para organizar minha rotina.
 - O sistema deve adicionar uma seção de "Tarefas Pendentes" no painel do estudante.
 - A lista deve agrupar atividades de todas as turmas em que o aluno está matriculado.
 - A ordenação padrão da lista deve ser cronológica.

## US05- Relatório das Avaliações
Como estudante, quero poder visualizar a nota geral obtida nas atividades já corrigidas, para que eu tenha uma visão clara do meu desempenho na matéria.
 - O aluno deve ter acesso a uma aba de “Notas” dentro da página de cada turma específica.
 - O sistema deve mostrar uma lista das atividades devolvidas com nota, mostrando a nota obtida e a nota máxima possível.
