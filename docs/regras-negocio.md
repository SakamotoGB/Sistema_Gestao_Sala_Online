# REGRA DE NEGOCIOS

## RN01- Criação de atividades
Apenas professores e administradores podem criar atividades.

## RN02- Entrega Pós-Prazo
Entregas realizadas após o prazo estipulado devem ser sinalizadas com atraso pelo sistema e podem sofrer penalidade na nota.

## RN03- Criação de Turmas
Apenas usuários que tenham o acesso de “Professor” ou “Administrador” têm permissão para criar novas turmas/sala de aula virtuais.

## RN04- Ingresso de Estudante
Um estudande só podera ser vinculado a uma turma se for adicionado manualmente pelo professor/administrador ou se inserir um código de convite gerado pelo sistema.

## RN05- Visualização de Notas
uma nota atribuida a alguma atividade só ficara visivel para o estudante se o professor executar a ação de devolver ou publicar determinada nota ou feedback

## RN06- Inalterabilidade Pós-Avaliação
Um estudante não é capaz de alterar os envios em atividades que já foram corrigidas pelo professor, ou seja que já estão como “Devolvidas” na plataforma.
