# Sistema de Gestão de Sala de Aula Online

Este repositório contém os artefatos de engenharia de software desenvolvidos como parte da **Avaliação Integrativa da disciplina de Modelagem e Projetos em Engenharia de Software** da Unijuí. O objetivo desta consultoria é conceber, modelar e documentar uma plataforma digital de gestão de sala de aula online, focada na gestão de turmas, atividades e avaliações.

## Estrutura do Repositório

*   **`docs/`**: Contém a documentação técnica, incluindo a Visão do Produto, identificação de Stakeholders, Regras de Negócio, User Stories, definição do MVP, documentação da Arquitetura e o Diário de Sprint.
*   **`bpmn/`**: Contém os arquivos do bpmn do fluxo de processos de negócios.
*   **`uml/`**: Contém o Diagrama de Casos de Uso representando as funcionalidades principais.
*   **`apresentacao/`**: Contém os slides da apresentação final.

## Ferramentas Utilizadas
*   **GitHub**: Utilizado para armazenamento de artefatos, documentação, controle de versões e evidências de colaboração.
*   **Gestor de Projetos**: GitHub Projects, utilizado para gerenciamento do backlog e organização do quadro Kanban.
